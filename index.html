This is an interactive data visualisation of the number π. It's a derivative work based on the 2013 π-day poster by Martin Krzywinski, http://mkweb.bcgsc.ca/pi/piday.

Each digit in the numbers are represented by a circle of a different color. Each circle contains a smaller circle colored according to the following digit. The colors are assigned as seen in the list above.

Tap or click on the number symbols in the top to choose a subject. Use the controls to change the number of rows, columns, radius of- and spacing between circles. Tap or click the image to toggle connecting lines.

Source code available on GitHub.
